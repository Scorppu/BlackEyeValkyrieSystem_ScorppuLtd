CSV Template Format

Your CSV file should include the following columns in this order:

primaryDrugId,interactingDrugId,severity,description

Example: "1,2,SEVERE,Causes severe blood pressure drop when taken together"

Note: The primaryDrugId and interactingDrugId must be valid drug IDs in the system.

Severity must be one of: MILD, MODERATE, SEVERE
