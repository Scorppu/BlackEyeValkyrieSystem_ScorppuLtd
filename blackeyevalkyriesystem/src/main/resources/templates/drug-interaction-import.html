CSV Template Format

Your CSV file should include the following columns:

primary_drug_name,interacting_drug_name1,interacting_drug_name2,...

Example: "Acetaminophen,Ibuprofen,Azithromycin,Lisinopril"

This imports interactions between Acetaminophen and each of the three drugs.

Note: All drugs must already exist in the system. Drug names are case-insensitive.
